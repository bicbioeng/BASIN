Installing and Using BASIN

2019-06-21

Introduction

The BASIN Image Analysis and Statistics Shiny app takes in flourescent microscope images of stained cell samples, separates them into distinct color components, and computes and plots statistics based on color intensities. The UI guides users to upload their images, tailor their experimental setup, visualize the results, and generate a custom report of their experiment.

Installation Instructions

BASIN incorporates the EBImage package for image processing and analysis found on the Bioconductor website. For more information on Bioconductor, please see their website at https://bioconductor.org. To install EBImage from Bioconductor using Bioconductor, run the following commands in R:

if (!requireNamespace("BiocManager", quietly = TRUE))
    install.packages("BiocManager") #installs Bioconductor
BiocManager::install("EBImage") #installs EBImage

Installing BASIN

Currently, BASIN is a private project, and not available on any public repository, so to install BASIN in RStudio, click on the Packages tab and click “Install” to open the Install Packages window. Under “Install From:” select “Package Archive File (.tgz; .tar.gz)” from the dropdown box, and if necessary click “Browse”. Then find the saved file “BASIN_(version number).tar.gz” on your computer, select it, and click “Install”. Alternatively, run the following command in R, replacing “/path/to/BASIN” with the full file path, including the file name.

install.packages("/path/to/BASIN", repos = NULL, type = "source")

After installing, attach the BASIN with the following command:

library(BASIN)
#> Loading required package: shiny

The BASIN Shiny app

The entire BASIN workflow is encompassed in the BASIN Shiny app, which takes individual image files or a folder of images with an accompanying .CSV file for analysis. The .CSV file should contain the file names of all images in the user-specified image folder, a condition type for each sample, and three columns for red, green, and blue metadata. Further instructions for uploading and analysis can be found in the app. To run the BASIN Shiny app, enter the following command in R:

runShinyBASIN()

Sample Data

A ‘figures’ folder can be found inside of the BASIN ‘vignettes’ folder that contains two sample images. Follow the instructions under the ‘Quick Run’ tab in the app to upload, analyze, and generate a report on these two images.
